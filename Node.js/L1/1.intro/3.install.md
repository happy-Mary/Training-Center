# Installation

* **Windows**: .msi / .exe
* **Mac OS**: .pkg / .tar.gz
* ***nix**: no babysitting

# IDEs

* IntelliJ WebStorm or IDEA
* Microsoft Visual Studio + Node.js Tools (Windows only)
* Cloud9
* Eclipse
* CodeRunner (OSX only)
* Sublime Text + Sublime Web Inspector

# Running

```
$ node app.js
```

# Debugging

## node-inspector

```
$ node inspect myscript.js
```

```
$ node --inspect myscript.js
```

Debugger listening on 127.0.0.1:9229.
